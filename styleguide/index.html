Paragraphs

CSS Selector

p

Sample Code

Rendered Element

The product I have chosen to create for P1 is a pair of wearable glasses with technology for artificial intelligence within them. While an individual is wearing these glasses they are able to see an interface that pops up in front their eyes. It will adapt to the environment around them and provide the user with information such as weather, alerts, virtual directions, news among with other statistics they can analyze. For example if the user is wearing the glasses and sees an animal, the glasses will run the image of the animal through its database and provide the user with what it is. Overall these glasses are intended to help the user in their day to day life and improve the efficiency and enhance the quality of daily tasks.
